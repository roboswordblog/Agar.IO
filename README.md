# Agar.IO
This is an Agar.IO clone made in python with pygame and sockets.

Remember, pygame cannot run on versoins 3.14, so you will have to downgrade to like 3.11 to run this.
This is multiplayer, so you will have to run the server before running the clients.
The code for the client will work for anyone on the same network as you, so that you can play with your classmates at school.
Or your family members.
New updates will be like:
1. A leaderboard
2. A chat
3. choosing your username
4. And more!
Remember, this is a project so that I could prove that I could make a agar.io clone in python better than in scratch.
